# Logo Placeholder

Este directorio debe contener el archivo `logo.png` para la aplicación PMLink Admin.

El logo debería ser:
- Formato: PNG
- Tamaño recomendado: 256x256 píxeles mínimo
- Fondo transparente (opcional)
- Colores que combinen con el tema naranja (#FFA500)

## Ubicación del logo

El archivo debe ubicarse en `public/pmlinkadmin/logo.png` para que sea accesible desde la aplicación.

Si no se proporciona un logo, la aplicación mostrará un placeholder con las iniciales "PM".
