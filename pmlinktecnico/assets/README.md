# Directorio de Assets

Este directorio debe contener los siguientes archivos de imagen:

- `icon.png` (1024x1024) - Icono de la aplicación
- `splash.png` (1242x2436) - Pantalla de splash
- `adaptive-icon.png` (1024x1024) - Icono adaptativo para Android
- `favicon.png` (48x48) - Favicon para web

Puedes generar estos assets usando:
- [Expo Icon and Splash screen generator](https://docs.expo.dev/guides/app-icons/)
- Cualquier herramienta de diseño gráfico

**Nota:** Estos archivos son necesarios para compilar la aplicación móvil.
